How to use Morphic

Overview

Morphic was developed as a tool to answer ecological questions as part of a study by Leighton et al. (2016)1.

Morphic is a work in progress, please contact the authors (gabi.leighton1@gmail.com, avoid3d@gmail.com) if you wish to use it.

Morphic is a tool that helps you analyse images it capture from Google Images. To use Morphic you first need to determine what information you need from the images. To gather this information you create a survey that will be filled out once for each of the images Morphic has captured.

Morphic is an open source project which is GPL v2.0 licensed. This means that you are free to use the source code as well as modify and redistribute it, as long as you also open source your own code. The authors request that you contact us (avoid3d@gmail.com) if you wish to use Morphic in your own project so that we can discuss fair attribution and citation.

1. Create a survey

The first step in using Morphic is to create a short relevant survey, to fill out per image. This survey will be used to capture the required information to answer your question.

You need to name your survey, and you can optionally add a descriptive comment which will be displayed next to the name to help you tell surveys apart.

Next, you need to add fields to your survey. A field is a question to be answered each time the survey is filled out. The field can be:

  • a text field which allows the user to type an answer
    • e.g.   
  • a radio buttons field which allows the user to choose one option from a list of options displayed side by side
    • e.g.    Apple Pear Orange
  • a select field which allows the user to select one option from a drop-down list
    • e.g.   
  • a location field which allows the user to select a location using a map provided by Google Maps API.
    • e.g. Where was this photograph taken?

Currently there is no way to edit a survey once it is created. This feature is under development. For now, please make sure that your survey is contains all neccesarry fields before clicking 'Create Survey'.

2. Add search results to the survey

An important first step is to install the morphic Chrome extension available here.

Now that you have a survey, you need to gather some images from Google Images. This is done by performing an Google Image search and using each image in the search results page. To do this you 'Add search' on the 'Searches' tab of the survey. When you first make a survey, you will also automatically be navigated to the 'Add search' form. One again you name the search and also optionally provide a descriptive comment. Next enter a 'Search Query'. This will be entered into Google Images when the search is performed. Morphic will actually perform a real image search on your behalf, so you can test out your search query using Google Images.

Once you have added a search you run it by clicking 'Run Search' which will open Google Images in a new tab. Don't do anything. Morphic will automatically close this tab and return you to your search results when the search is finished.

3. Fill out the survey

Now that you have images associated with your survey, you can begin to analyse them by navigating to the 'Search Results' tab and clicking on a search result and completing the survey for that image. Once you have filled in survey you can tag it as one of 'Done', 'Revisit' or 'Not Usable'. To save your additions, click Save or use one of the macros on the navigation bar, which tag the survey, save the fields and also navigate to the next image.

4. Exporting the raw data

Exporting your data is the last step. Once you are satisfied you have gathered enough results go to your survey and navigate to the 'Export Results' tab. Then click the 'Export Results' button which will start a download in your browser. This file will be a newline delimited csv with your suvery field labels as the header and one row per search result.

1. Gabriella Leighton, Pierre Hugo, Alexandre Roulin and Arjun Amar (2016) 'Just Google it: assessing the use of Google Images to describe geographical variation in visible traits of organisms.' Methods in Ecology and Evolution
